如何获取 Google Translate API Key？
  1. 前往 console.cloud.google.com，登录 Google 账号
  2. 顶部选择或新建一个项目
  3. 左侧 → 「API 和服务」→「库」，搜索 Cloud Translation API 并启用
  4. 左侧 → 「API 和服务」→「凭据」→「创建凭据」→「API 密钥」
  5. 建议「限制密钥」→ API 限制设为 Cloud Translation API，防止滥用
  6. 复制密钥粘贴到上方输入框，点击保存

费用：每月前 50 万字符免费，超出约 ¥14 / 百万字符
