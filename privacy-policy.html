新词本 Chrome 扩展隐私政策

生效日期：2026-03-19

本隐私政策适用于 Chrome 扩展“新词本 - 英文生词收藏”。

1. 我们收集哪些信息

为实现扩展核心功能，本扩展可能处理以下信息：

  • Google 账号基础信息：包括用户 ID、邮箱地址、显示名称、头像。
  • 用户主动保存的生词数据：包括单词本身、收藏次数、收藏时间。
  • 用户主动触发收藏时的上下文信息：包括选中单词所在句子、页面标题、页面 URL。
  • 用户设置数据：包括主题、排序方式、翻译服务选择。
  • 用户主动填写的翻译配置：如果用户选择 Google 翻译模式，扩展会保存用户自行填写的 Google Translate API Key。

本扩展不会在用户未主动操作时自动提取整页内容，也不会采集与生词收藏无关的网页数据。

2. 这些信息如何被使用

上述信息仅用于以下目的：

  • 让用户使用 Google 账号登录扩展。
  • 将用户的生词本同步到其个人 Firebase Firestore 数据库。
  • 在弹窗中展示用户收藏的单词、上下文、次数和翻译结果。
  • 在用户主动点击翻译时调用第三方翻译或词典服务。
  • 保存用户的个性化设置与翻译偏好。

3. 数据存储位置

数据可能存储在以下位置：

  • Chrome 本地或同步存储：用于保存登录状态、主题、翻译设置、用户填写的 API Key 等。
  • Firebase Authentication：用于处理登录认证。
  • Firebase Firestore：用于保存用户个人生词本数据。

扩展开发者不会在上述服务之外额外建立独立服务器来存储你的生词数据。

4. 第三方服务

本扩展会在用户使用相关功能时与以下第三方服务通信：

  • Google Identity / Chrome Identity：用于 Google 登录。
  • Firebase Authentication：用于身份认证。
  • Firebase Firestore：用于数据同步与存储。
  • Free Dictionary API：用于获取英文单词释义、近义词、反义词。
  • MyMemory Translation API：当用户选择免费翻译模式时，用于翻译文本。
  • Google Cloud Translation API：仅当用户主动选择 Google 翻译模式并填写自己的 API Key 时使用。

这些第三方服务受其各自隐私政策和服务条款约束。

5. 数据共享情况

除为实现上述核心功能而向相关第三方服务发送必要请求外，本扩展不会出售、出租或主动向无关第三方共享用户个人数据。

本扩展不将用户数据用于广告投放、用户画像或数据经纪用途。

6. 用户控制与删除

用户可以通过以下方式控制自己的数据：

  • 在扩展中删除单个生词或清空全部生词。
  • 在 Chrome 中卸载扩展，删除本地存储数据。
  • 在 Firebase 项目中删除相关用户数据。
  • 如果用户填写过 Google Translate API Key，可在扩展设置中删除或覆盖该 Key。

7. 数据安全

我们尽力通过 Firebase 认证、Firestore 安全规则及 Chrome 扩展存储机制保护数据安全。但任何互联网传输或电子存储方式都无法保证绝对安全。

8. 儿童隐私

本扩展面向一般用户，不针对儿童提供专门服务。

9. 隐私政策更新

如果本隐私政策发生重大变化，我们将通过扩展更新版本或项目页面更新内容。

10. 联系方式

如果你对本隐私政策有疑问，建议通过 Chrome Web Store 页面中提供的开发者联系邮箱与开发者联系。
